## external_annotations

This directory contains external data from genome annotation from TAIR12: CDS sequences, TE library, and annotation GFF.
> Leonore Reiser, Alyssa Proia, Erica Bakker, Shabari Subramaniam, Kartik Khosa, Swapnil Sawant, Xingguo Chen, Trilok Prithvi, Tanya Z Berardini, Recent major changes to TAIR: updates to the database, website, and Arabidopsis genome, Genetics, Volume 232, Issue 4, April 2026, iyaf248, https://doi.org/10.1093/genetics/iyaf248
