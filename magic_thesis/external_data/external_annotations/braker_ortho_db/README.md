## braker_ortho_db

This directory contains external annotation data used for input into BRAKER2 (combination of Viridiplantae orthologous proteins and Arabidopsis thaliana proteins from TAIR12).

>Fredrik Tegenfeldt, Dmitry Kuznetsov, Mosè Manni, Matthew Berkeley, Evgeny M Zdobnov, Evgenia V Kriventseva, OrthoDB and BUSCO update: annotation of orthologs with wider sampling of genomes, Nucleic Acids Research, 2024; gkae987, https://doi.org/10.1093/nar/gkae987


>Leonore Reiser, Alyssa Proia, Erica Bakker, Shabari Subramaniam, Kartik Khosa, Swapnil Sawant, Xingguo Chen, Trilok Prithvi, Tanya Z Berardini, Recent major changes to TAIR: updates to the database, website, and Arabidopsis genome, Genetics, Volume 232, Issue 4, April 2026, iyaf248, https://doi.org/10.1093/genetics/iyaf248
