# Data Science Honors Thesis

_Megan Tandean_

This project aims to assemble the genomes of 8 accessions of _Arabidopsis thaliana_ that have been used to create a multi-parent advanced inter-cross (MAGIC) population, and to study the genetic and climatic variation between these ecotypes. 

## Directory Structure

This repo contains the pipelines to run the project, the full scripts used, and directory structure that would be created by running the code.  

*Note: the file locations in the early code currently may not match the directory because the structure was modified. I will update them to match as the directory structure is finalized*

### Main Folders
|  Folder | Description |
|-----|-----|
| `general_pipelines`  | example code for all individual parts of the project with input  |
| `magic_thesis`  | directory for the analysis of 8 MAGIC paraent genomes |
| `quality_check` | directory for checking the quality of the 8 MAGIC parent genomes with the genomes published in "A pan-genome of 69 Arabidopsis thaliana accessions reveals a conserved genome structure throughout the global species range" (Lian et al., 2024) |

### Subfolders in *magic_thesis*
| Folder | Description |
|-----|-----|
| `external_data`  | externally sourced datasets |
| `intermediate_data`  | generated data necessary for downstream pipelines |
| `log_slurm`  | log files generated by slurm jobs |
| `meta_info`  | metadata of 8 parental accessions |
| `raw_data`  | PacBio Revio HiFi sequencing data of 8 parental accessions |
| `results`  | generated figures and tables |
| `scripts`  | exact code used to run pipelines on all 8 parental accessions |

### Subfolders in *quality_check*
| Folder | Description |
|-----|-----|
| `data`  | raw reads, assemblies, and intermediate data by source  |
| `log_slurm`  | log files generated by slurm jobs |
| `results`  | generated tables |
| `scripts`  | exact code used to run pipelines on all accessions |
